04. 웹팩(webpack) React 연동

2018.03.25 in Frontend

react를 연동해보자, 이전 포스팅과 연결된다.

npm i react react-dom babel-preset-react

  • src/main.js
  • src/index.html
  • src/app.js
  • .babelrc

위와 같이 리액트 바벨 프리셋을 추가해주자!

소스변경시에도 리액트의 state 값을 유지시켜주는 핫모듈도 있지만, 왜 때문인지 연동이 안되서 일단 보류 한다.
