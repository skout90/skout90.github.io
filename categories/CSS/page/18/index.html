CSS 상대 크기

2017.07.24 in CSS

CSS 상대적인 크기에 대해 알아본다.

em

  • 최근에 정의한(부모 요소) font-size를 1em 으로 함, 계속해서 누적됌.
  • 없으면 브라우저 기본값을 따름
  • 9pt = 16px = 1em

REM

  • 최상위 브라우저 기본 폰트 크기를 1rem으로 함, 누적되지 않음.

vh, ve

브라우저 윈도우 크기의 1/100 크기를 “1”로 함

  • 2000 x 1000 px 크기의 윈도우라면

1vh = 10px

1vw = 20px
