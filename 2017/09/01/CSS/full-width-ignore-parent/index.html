부모에 상관없이 width 100% 주는 법

2017.09.01 in CSS

상위 요소의 넓이나 높이에 관계없이, 100vw는 브라우저 화면의 넓이,

100vh는 브라우저 화면의 높이를 뜻함.

vw 는 IE9 이상 및 그 외 브라우저에서 지원해줌.

calc 함수를 써서, left를 화면의 0의 위치에서 시작하게끔 함. (IE8 이상)
