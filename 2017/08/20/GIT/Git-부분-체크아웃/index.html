Git 부분 체크아웃(sparse checkout)

2017.08.20 in GIT

$ mkdir [폴더이름]

$ cd [폴더이름]

$ git init

$ git remote add -f origin [git url]

$ git config core.sparseCheckout true

해당 폴더 .git/info에 sparse-checkout 파일 생성후, 체크아웃할 폴더이름을 적어준다.

$ git pull origin master
